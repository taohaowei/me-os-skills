# 健康记录员子系统

> **免责声明**：本子系统**不提供任何医疗建议、诊断或治疗方案**，仅用于记录与整理用户本人及家属的健康信息。任何健康决策请咨询专业医疗人员。如遇紧急情况或心理危机，请参见 `config/crisis-hotlines.{{locale}}.md`。

## 这是什么

健康记录员是一个**纯记录、不诊断**的个人健康档案子系统。它帮助你把分散的健康信息——症状、就医、用药、体检、生活方式——结构化地沉淀下来，便于回顾、对比与就诊时调取。

## 核心原则

- **只记录，不诊断**：不给出医疗结论、用药或治疗建议。
- **可追溯**：每条记录带日期与来源，`timeline.md` 倒序索引全部事件。
- **隐私优先**：本仓为公开模板，真实信息只存于 `$ME_HOME/health/`，不入公共仓。
- **模糊标注**：信息不全时统一标 `[待确认]`，不臆测。

## 目录结构

数据根：`$ME_HOME/health/`

- `profile.md`：个人健康档案（不常变信息）
- `timeline.md`：全局倒序索引（核心入口）
- `conditions/`：症状 / 病情跟踪
- `visits/`：就医记录
- `medications/`：用药记录（当前 + 历史）
- `checkups/`：体检报告
- `doctors/`：医生档案与评价
- `insurance/`：保险与报销
- `lifestyle/`：运动 / 饮食 / 睡眠
- `notes/`：临时笔记

## 如何使用

1. 将本目录模板复制到 `$ME_HOME/health/`（默认 `~/.me/health/`）。
2. 去掉文件名中的 `.template`，按各文件结构填写，占位符替换为真实内容。
3. 录入信息时遵循 `CLAUDE.md`（健康记录员协议）：每次新增同步更新 `timeline.md`。

详细录入规范见同目录 `CLAUDE.md`。
